VN Font Converter

Version: 

Date: September 2014

=========


ArcGIS Toolbox convert font Vietnamese for shapefile/ feature class


Công cụ chuyển đổi mã font cho shapefile/ feature class:

 chuyển giữa các bảng mã TCVN3-VNIWindows-Unicode & chuyển về không dấu,

 kèm theo lựa chọn chuyển sang chữ hoa, chữ thường, hoa đầu câu, hoa mỗi từ.

Hướng dẫn sử dụng:

- Chạy file Shapefile_Unicode_*.reg tương ứng với phiên bản ArcGIS đang dùng.

- Khởi động ArcCatalog/ ArcMap, vào ArcToolbox --> Add Toolbox --> chọn file "VNFont Converter.tbx".
 Chọn Save Settings --> To Default nếu muốn ArcGIS nạp sẵn công cụ VNConvert khi khởi động.

- Khởi động nhanh: Trong ArcCatalog tìm đến thư mục chứa "VNFont Converter.tbx" va "VNFontConverter.py".
Nhấp chuột vào dấu "+" để mở "Chuyển đổi mã fone" để sử dụng.
